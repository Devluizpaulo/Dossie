# **App Name**: BMV Dossier Viewer

## Core Features:

- Dossier Display: Display the BMV Dossiê de Avaliação in a structured and user-friendly format.
- Table of Contents: Generate and display a dynamic table of contents for easy navigation within the dossier.
- Search Functionality: Implement a search feature to quickly find specific information within the dossier.
- Theme Toggle: Allow users to switch between light and dark themes for optimal viewing experience.
- Link Management: All links within the documentation must be live, and correctly point to: https://backoffice.bmv.global, https://legado-backoffice.bmv.global, https://multiledgers.com.

## Style Guidelines:

- Primary color: Deep blue (#1E3A8A), evokes trust and professionalism.
- Background color: Light gray (#F0F3F5), provides a neutral and clean backdrop.
- Accent color: Orange (#EA580B), highlight key interactive elements.
- Body and headline font: 'Inter' (sans-serif) for a modern and readable style.
- Use simple and clear icons to represent navigation items and actions.
- Maintain a clean and organized layout with clear sections and headings for easy readability.
- Use subtle transitions and animations for interactive elements to enhance user experience.
